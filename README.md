# 🎬 AhnaJak - Movie App

A React Native movie app built with Expo

## Features
- Browse movies by category
- Search movies
- Movie detail page
- Login with API

## Tech Stack
- React Native + Expo
- JavaScript
- REST API (DummyJSON)
- Expo Router
